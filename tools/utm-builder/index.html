What is a UTM Builder?

      

        A UTM Builder adds tracking parameters to a URL so analytics tools can attribute clicks to the right channel and campaign.
        It‚Äôs the clean way to measure performance for social profiles, video descriptions, newsletters, ads, and partner mentions.
      


      

        

          
1) Paste the destination URL

          
Your link stays intact ‚Äî existing parameters are preserved.

        

        

          
2) Add source / medium / campaign

          
Those 3 fields are the core for most tracking. Use content/term for variants.

        

        

          
3) Copy your trackable link

          
Use it in bio links, video descriptions, newsletters, ads, and referrals.

        

      


      
How to name UTMs (clean + consistent)

      

        Keep values short, consistent, and readable. Example: utm_source=instagram, utm_medium=social,
        utm_campaign=tools-directory. Use utm_content when you have multiple links or variants.

FAQ

      

        

          
Do Clickoz tools upload my URL or data?

          
No. This tool runs locally in your browser. Your inputs stay on your device.

        

        

          
Which UTM fields are required?

          
Usually source, medium, and campaign. Term/content are optional but useful for ads and A/B tests.

        

        

          
Can I paste a URL without https?

          
Yes. The tool assumes https automatically so the final link is valid.

        

        

          
Does it keep my existing parameters?

          
Yes. Existing query params stay. UTMs are added (or updated if already present).
